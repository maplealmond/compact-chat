# Compact Chat

* Makes the siderail chat render with a black background, like it was in the v12 or earlier experience.  Combine with the Pip Notification setting for closer to the classic UI experience.
* User config setting to make the siderail buttons more compact
* User config settings to hide certain unused tabs if desired (e.g. no need to show cards if your system doesn't use them)
